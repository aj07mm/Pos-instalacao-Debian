# Pos-instalacao-Debian
Script simples para instalar de programas após a instalação do Debian

__Programas que o Script Instala:__
- Atualizar o sistema                           
- Adiconar non-free ou contrib/non-free         
- Ufw                                  
- Build-essential                      
- Atom                                 
- Telegram                             
- Ruby Gems                            
- Jekyll                               
- NodeJs
- Compose
- LAMP
- sudo                              
- Vim                                 
- Tmux                                
- Inkscape                            
- Gimp                                
- Audacity                            
- Kdenlive                            
- Vlc                                
- Chromium                           
- Terminal do XFCE

__Como usar:__
```bash
bash debian.sh
```
